
# Лабораторная работа №17

## Методические указания

При выполнении лабораторной работы следует:

 - [X] T.1. Написать функцию для ввода текущего количества строк/столбцов массива с проверкой корректности вводимых данных;

 - [X] T.2. Написать функции для заполнения массива исходными данными. Заполнение массива должно быть реализовано с использованием механизма перегрузки функций (в зависимости от желания пользователя): - с клавиатуры (параметрами являются указатель – имя массива и количество строк, столбцов); - случайными числами (параметрами являются указатель – имя массива, диапазон и количество строк, столбцов); - из файла (параметрами являются указатель – имя массива, имя файла и количество строк, столбцов, предусмотреть случай, когда в файле меньше чисел, чем заявлено при объявлении массива). Выбор способа заполнения массива реализовать в функции `main()`;

 - [X] T.3. Написать функцию, выводящую элементы массива на экран;

 - [X] T.4. Написать функции для обработки массивов в соответствии с заданием своего варианта. Для каждого пункта задания должна быть написана как минимум одна функция обработки (с необходимыми параметрами). Задания выполняются по порядку. Дополнительные массивы использовать не разрешается;

 - [X] T.5. Результат выводить на экран и в текстовый файл.

## Варианты заданий для лабораторной работы

`Вариант 3`

 - [X] I.1. Найти сумму и количество элементов каждого столбца с заданным условием: данные элементы положительные и лежат ниже второстепенной диагонали. Результат хранить в одномерных массивах;
 
 - [X] I.2. Определить, есть ли в данном массиве строка, состоящая только из положительных элементов.
