
# Лабораторная работа №11

## Задание:

Необходимо реализовать программный код, позволяющий найти сумму двух дробей, результат представить в виде несократимой дроби. При выполнении лабораторной работы следует: 
 - [X] T.1. Написать функцию, обеспечивающую ввод дроби с клавиатуры и проверку на корректность;
 - [X] T.2. Написать функцию, возвращающую наибольший общий делитель двух натуральных чисел;
 - [X] T.3. Написать функцию сокращения дроби;
 - [X] T.4. Написать функцию, вычисляющую сумму двух дробей, Возвращаемая дробь должна быть несократимой;
 - [X] T.5. Написать функцию подсчитывающую количество цифр в числе;
 - [X] T.6. Написать функцию вывода дроби на экран (длина черты между числителем и знаменателем должна быть равна количеству цифр большего из выводимых чисел);
 - [X] T.7. Вызывая функцию нахождения суммы дробей нужное количество раз, вывести результаты на экран.

## Индивидуальное задание:

 - [ ] I.1. вычислить сумму первых двух слагаемых как сумму обычных дробей;
 - [ ] I.2. при необходимости полученную дробь сократить;
 - [ ] I.3. до тех пор, пока не вычислена вся сумма, прибавлять к имеющейся дроби очередное слагаемое и при необходимости сокращать.

`Вариант 3`:
 * Для последовательности $а_1 = 1, a_{n+1} ={\frac {a_n + 1} {(1 + a_n)} }$ составить программу печати $n{\text{-го}}$ элемента в в виде несократимой дроби.
