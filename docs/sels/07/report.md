## 1. Задача №1

### 1.1. Постановка задачи

Написать функцию для вычисления приближенной суммы функционального ряда с заданной точностью при фиксированном значении $x$. Функция должна возвращать значение суммы и количество выполненных итераций.

Коэффициенты ряда вычисляются по формуле:

$$a_n = (-1)^n \frac{2^n}{n!}$$

Требуется исключить использование функции `pow()`, условной операции `?:` и повторного вычисления факториала и степеней на каждом шаге итерации.

### 1.2. Входные данные

 - $x$ – значение аргумента функции;
 - $\varepsilon$ – заданная точность вычисления.

### 1.3. Выходные данные

 - $S$ – приближённая сумма ряда;
 - $n$ – количество выполненных итераций.

### 1.4. Математическая модель задачи

#### 1.4.1. Основные понятия

 - $x$ – аргумент функционального ряда;
 - $a_n$ – коэффициент ряда;
 - $S$ – сумма ряда;
 - $\varepsilon$ – точность вычисления;
 - $n$ – номер итерации.

#### 1.4.2. Формульная зависимость

$$
S = \sum_{n=1}^{\infty} a_n x^n
$$

где

$$
a_n = (-1)^n \frac{2^n}{n!}
$$

Рекуррентное вычисление очередного члена ряда:

$$
a_n x^n = a_{n-1} x^{n-1} \cdot \frac{-2x}{n}
$$

while (abs(term) >= eps) {
  sum += term;
  iterations++;

  n++;
  coefficient = variantCoefficient(n, coefficient);
  xPower     *= x;
  term        = coefficient * xPower;
}

cout << "Приближенная сумма ряда: "         << sum        << endl;

## 2. Задача №2

### 2.1. Постановка задачи

Составить рекурсивную функцию для перевода натурального числа в $p$-ичную систему счисления, где $2 \le p \le 9$.

Также требуется реализовать рекурсивную функцию вычисления многочлена Чебышёва:

$$
T_0(x)=1
$$

$$
T_1(x)=x
$$

$$
T_n(x)=2xT_{n-1}(x)-T_{n-2}(x)
$$

### 2.2. Входные данные

 - $N$ – натуральное число;
 - $p$ – основание системы счисления;
 - $x$ – аргумент многочлена Чебышёва;
 - $n$ – номер многочлена.

### 2.3. Выходные данные

 - число $N$ в $p$-ичной системе счисления;
 - значение многочлена Чебышёва $T_n(x)$.

### 2.4. Математическая модель задачи

#### 2.4.1. Основные понятия

 - $N$ – исходное число;
 - $p$ – основание системы счисления;
 - $T_n(x)$ – многочлен Чебышёва порядка $n$.

#### 2.4.2. Формульная зависимость

Перевод числа в систему счисления выполняется рекурсивным делением:

$$
N = p \cdot q + r
$$

где:

 - $q$ – целая часть от деления;
 - $r$ – остаток от деления.

Многочлены Чебышёва:

$$
T_n(x)=2xT_{n-1}(x)-T_{n-2}(x)
$$

### 2.7. Код программы

```cpp
/**
 * @author ${author_full}
 * @brief  Самостоятельная работа 06
 */

/* Включение необходимых библиотек
  (см. репозиторий автора ${repository} ) */
#include "index.hpp"

auto toBase = [&](auto self, int n, int base) -> void {
  if (n >= base) {
    self(self, n / base, base);
  }

  cout << n % base;
};

auto chebyshev = [&](auto self, int n, double x) -> double {
  if (n == 0) {
    return 1.0;
  }

  if (n == 1) {
    return x;
  }

  return 2 * x * self(self, n - 1, x) - self(self, n - 2, x);
};

int number;
int base;

cout << "Введите натуральное число: "; cin  >> number;

cout << "Введите основание системы счисления (2-9): "; cin  >> base;

cout << "Число в новой системе счисления: ";
toBase(toBase, number, base);
cout << endl;

int    n;
double x;

cout << "Введите номер многочлена Чебышёва n: "; cin  >> n;

cout << "Введите значение x для вычисления многочлена: "; cin  >> x;

double result = chebyshev(chebyshev, n, x);

cout << "Значение многочлена Чебышёва Tn(x): " << result << endl;
```
