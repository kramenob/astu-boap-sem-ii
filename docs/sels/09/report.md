## 1. Задача №1

### 1.1. Постановка задачи

Заменить в заданной строке все вхождения символа Ch1 на символ Ch2. Значения символов вводятся с клавиатуры.

### 1.2. Входные данные

- строка текста;
- символ Ch1 (заменяемый символ);
- символ Ch2 (новый символ).

### 1.3. Выходные данные

- изменённая строка после замены всех вхождений Ch1 на Ch2.

### 1.4. Математическая модель задачи

#### 1.4.1. Основные понятия

- $S$ — исходная строка;
- $Ch1$ — символ, который необходимо заменить;
- $Ch2$ — символ, на который производится замена.

#### 1.4.2. Формульная зависимость

$$
S' = f(S)
$$

где:

- если $S[i] = Ch1$, то $S'[i] = Ch2$;
- иначе $S'[i] = S[i]$.

### 1.7. Код программы

```cpp
/**

 * @brief  Самостоятельная работа 09
 */

/* Включение необходимых библиотек
  (см. репозиторий автора ${repository} ) */

string text;

char ch2;

cin.ignore(numeric_limits<streamsize>::max(), '\n');
cout << "Введите строку: ";
getline(cin, text);

cout << endl << "Введите символ, который нужно заменить: ";
cin  >> ch1;

cout << "Введите символ, на который нужно заменить: ";
cin  >> ch2;

for (int i = 0; i < text.length(); i++) {
  if (text[i] == ch1) {
    text[i] = ch2;
  }
}

cout << endl << "Результат замены символов: " << text << endl;
```

## 2. Задача №2

### 2.1. Постановка задачи

В тексте слова записаны в зашифрованном виде (каждое слово записано наоборот). Необходимо расшифровать сообщение, восстановив исходные слова.

### 2.2. Входные данные

- строка текста, содержащая зашифрованные слова.

### 2.3. Выходные данные

- строка текста с расшифрованными словами.

### 2.4. Математическая модель задачи

#### 2.4.1. Основные понятия

- $S$ — исходный текст;
- $W_i$ — слово в тексте.

#### 2.4.2. Формульная зависимость

$$
W'_i = reverse(W_i)
$$

### 2.7. Код программы

```cpp
/**
 * @author ${author_full}
 * @brief  Самостоятельная работа 09
 */

/* Включение необходимых библиотек
  (см. репозиторий автора ${repository} ) */
#include "index.hpp"

string reverseWord(string word) {
  reverse(word.begin(), word.end());
  return word;
}

string encryptedText;

cin.ignore(numeric_limits<streamsize>::max(), '\n');
cout << "Введите зашифрованный текст: ";
getline(cin, encryptedText);

stringstream stream(encryptedText);
string word;

cout << endl << "Расшифрованное сообщение: ";

while (stream >> word) {
  cout << reverseWord(word) << " ";
}
```
