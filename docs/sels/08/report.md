## 1. Задача №1

### 1.1. Постановка задачи

Заполнить файл `Input.txt` целыми числами. Длина последовательности должна составлять от 10 до 30 элементов. Последовательность может вводиться пользователем или генерироваться случайным образом.

Требуется найти среднее арифметическое положительных элементов данной последовательности.

### 1.2. Входные данные

 - последовательность целых чисел из файла `Input.txt`.

### 1.3. Выходные данные

 - среднее арифметическое положительных элементов последовательности.

### 1.4. Математическая модель задачи

#### 1.4.1. Основные понятия

 - $a_i$ – элемент последовательности;
 - $n$ – количество элементов последовательности;
 - $S$ – сумма положительных элементов;
 - $k$ – количество положительных элементов.

#### 1.4.2. Формульная зависимость

$$
\overline{x} = \frac{S}{k}
$$

где:

$$
S = \sum a_i, \quad a_i > 0
$$

cout << "Сгенерированная последовательность: ";

for (int i = 0; i < length; i++) {
  int number = rand() % 101 - 50;

  if (number > 0) {
    positiveSum += number;
    positiveCount++;
  }
}

cout << endl << endl;

inputFile.close();

if (positiveCount > 0) {
  double average = positiveSum / positiveCount;
  cout << "Среднее арифметическое положительных элементов: " << average << endl;
} else {
  cout << "Положительные элементы отсутствуют." << endl;
}

## 2. Задача №2

### 2.1. Постановка задачи

Обработать последовательность чисел без использования массива.

В выходную последовательность необходимо записать только те числа, которые меньше суммы своих соседей:

 - первое число записывается, если оно меньше второго;
 - внутренний элемент записывается, если он меньше суммы соседних элементов;
 - последнее число записывается, если оно меньше предпоследнего.

### 2.2. Входные данные

 - последовательность целых чисел.

### 2.3. Выходные данные

 - обработанная последовательность чисел.

### 2.4. Математическая модель задачи

#### 2.4.1. Основные понятия

 - $a_i$ – текущий элемент последовательности;
 - $a_{i-1}$ – предыдущий элемент;
 - $a_{i+1}$ – следующий элемент.

#### 2.4.2. Формульная зависимость

Элемент включается в результирующую последовательность, если:

$$
a_i < a_{i-1} + a_{i+1}
$$

### 2.7. Код программы

/**
 * @author Болотин Максим Владимирович
 * @brief  Самостоятельная работа 08
 */

/* Включение необходимых библиотек
  (см. репозиторий автора github.com/kramenob/astu-boap-sem-ii ) */
#include "index.hpp"

ifstream inputFile(iFile);

int previous;
int current;
int next;

if (!(inputFile >> previous)) {
  cout << "Файл Input.txt пуст." << endl;
  return;
}

if (!(inputFile >> current)) {
  cout << "Недостаточно данных для обработки." << endl;
  return;
}

cout << "Результат обработки последовательности: ";

if (previous < current) {
  cout << previous << " ";
}

while (inputFile >> next) {
  if (current < previous + next) {
    cout << current << " ";
  }

  previous = current;
  current = next;
}

if (current < previous) {
  cout << current << " ";
}

cout << endl;

inputFile.close();
```

## 3. Задача №3

### 3.1. Постановка задачи

Получить исходную последовательность чисел из файла `Input.txt` и записать результат обработки в файл `Output.txt`.

Требуется найти максимальное чётное число и его порядковый номер. При отсутствии чётных чисел необходимо вывести соответствующее сообщение.

### 3.2. Входные данные

 - последовательность целых чисел из файла `Input.txt`.

### 3.3. Выходные данные

 - максимальное чётное число;
 - порядковый номер максимального чётного числа.

### 3.4. Математическая модель задачи

#### 3.4.1. Основные понятия

 - $a_i$ – элемент последовательности;
 - $max$ – максимальное чётное число;
 - $i$ – номер элемента.

#### 3.4.2. Формульная зависимость

$$
max = \max(a_i), \quad a_i \bmod 2 = 0
$$

/**
 * @author Болотин Максим Владимирович
 * @brief  Самостоятельная работа 08
 */

/* Включение необходимых библиотек
  (см. репозиторий автора github.com/kramenob/astu-boap-sem-ii ) */
#include "index.hpp"

ifstream inputFile(iFile);
ofstream outputFile(oFile);

int number;
int index = 0;

int maxEven      =  0;
int maxEvenIndex = -1;

bool hasEven = false;

while (inputFile >> number) {
  outputFile << number << " ";

  if (number % 2 == 0) {
    if (!hasEven || number > maxEven) {
      maxEven      = number;
      maxEvenIndex = index;
      hasEven      = true;
    }
  }

  index++;
}

inputFile.close();
outputFile.close();

if (hasEven) {
  cout << "Максимальное чётное число: " << maxEven      << endl;
  cout << "Порядковый номер: "          << maxEvenIndex << endl;
} else {
  cout << "Чётные числа отсутствуют." << endl;
}
```
