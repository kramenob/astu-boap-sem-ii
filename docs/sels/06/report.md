## 1. Задача №1

## 1.1. Постановка задачи

Вычислить значение неэлементарной функции $y = f(x)$. Реализовать решение несколькими способами:
 - полное ветвление;
 - краткая форма ветвления с использованием составных логических выражений;
 - с использованием условной операции.

Используя оператор–переключатель, реализовать выбор способа решения.  

### 1.2. Входные данные

 - $x$ – значение из множества натуральных чисел;

### 1.3. Выходные данные

 - $y$ – значение из множества действительных чисел.

### 1.4. Математическая модель задачи

#### 1.4.1. Основные понятия

 - $x$ – аргумент выражений, область допустимых значений;
 - $y$ – значение функции.

#### 1.4.2. Формульная зависимость

$$y=\begin{cases}
   \frac{ \cos{ 2 } }{ x }    &\text{при }      x < -5 \\
   x \sin{ x }                &\text{при } -5 ≤ x ≤  2 \\
   \frac{ x + 10 }{ 2 + x^2 } &\text{при }  2 < x < 3 \\
   x + \sqrt[3]{x}            &\text{при }      x ≥  3
\end{cases}$$

## 2. Задача №2

## 2.1. Постановка задачи

Составить программу, определяющую результат гадания на ромашке («любит–не любит–плюнет–поцелует–к сердцу прижмёт–к чёрту пошлёт»), используя количество лепестков ромашки $N$.

### 2.2. Входные данные

 - $N$ – количество лепестков ромашки, натуральное число от 10 до 50.

### 2.3. Выходные данные

 - результат гадания.

### 2.4. Математическая модель задачи

#### 2.4.1. Основные понятия

 - $N$ – количество лепестков ромашки;
 - остаток от деления $N \bmod 6$ определяет результат гадания.

#### 2.4.2. Формульная зависимость

$$
R =
\begin{cases}
0 \rightarrow \text{«любит»} \\
1 \rightarrow \text{«не любит»} \\
2 \rightarrow \text{«плюнет»} \\
3 \rightarrow \text{«поцелует»} \\
4 \rightarrow \text{«к сердцу прижмёт»} \\
5 \rightarrow \text{«к чёрту пошлёт»}
\end{cases}
$$

где $R = N \bmod 6$.

### 2.5. Алгоритм решения задачи

1. Ввести количество лепестков $N$.
2. Найти остаток от деления $N$ на $6$.
3. Определить результат гадания по полученному значению.
4. Вывести результат на экран.

## 3. Задача №3

## 3.1. Постановка задачи

Вычислить значение функции $S = f(x)$ с использованием разложения в степенной ряд для заданной точности $\varepsilon$.

### 3.2. Входные данные

 - $x$ – значение аргумента;
 - $\varepsilon$ – точность вычисления, где $0 < \varepsilon < 1$.

### 3.3. Выходные данные

 - $S$ – значение суммы ряда.

### 3.4. Математическая модель задачи

#### 3.4.1. Основные понятия

 - $x$ – аргумент функции;
 - $\varepsilon$ – требуемая точность вычисления;
 - $n$ – номер члена ряда.

#### 3.4.2. Формульная зависимость

$$S = x - \frac{ x^3 }{ 3! } + ... + ( -1 )^n \frac{ x^{ 2n + 1 } }{ ( 2n + 1 )! }$$

### 3.5. Алгоритм решения задачи

1. Ввести значения $x$ и $\varepsilon$.
2. Вычислять члены ряда.
3. Суммировать члены ряда до тех пор, пока модуль очередного члена не станет меньше $\varepsilon$.
4. Вывести значение суммы ряда.

### 3.6. Тестовые наборы данных

| Входные данные                    | Результат ручной проверки | Результат работы системы | Комментарий |
|-----------------------------------|---------------------------|--------------------------|-------------|
| $x = 0.5$, $\varepsilon = 0.001$  | $S \approx 0.48$          | $S \approx 0.48$         | Успех       |
| $x = 1$,   $\varepsilon = 0.0001$ | $S \approx 0.84$          | $S \approx 0.84$         | Успех       |
| $x = 2$,   $\varepsilon = 0.001$  | $S \approx 0.91$          | $S \approx 0.91$         | Успех       |

### 3.7. Код программы

```cpp
/**
 * @author Болотин Максим Владимирович
 * @brief  Самостоятельная работа 06
 */

/* Включение необходимых библиотек
  (см. репозиторий автора github.com/kramenob/astu-boap-sem-ii ) */
#include "index.hpp"

double x       = 0.0;
double epsilon = 0.0;

cout << "Введите x: ";
cin  >> x;

cout << "Введите epsilon (0 < epsilon < 1): ";
cin  >> epsilon;

if (epsilon <= 0 || epsilon >= 1) {
  cout << "Ошибка. Некорректное значение epsilon." << endl << endl;
  return;
}

double sum  = 0.0;
double term = x;
int    n    = 0;

while (fabs(term) >= epsilon) {
  sum += term;
  n++;
  term *= (-1.0) * x * x / ((2 * n) * (2 * n + 1));
}

cout << endl << "Сумма ряда S = " << sum << endl;
```

## 4. Задача №4

## 4.1. Постановка задачи

Затабулировать функцию $Z = f(x, y)$ с шагами $\Delta x$ и $\Delta y$ внутри заданного прямоугольника и области существования функции.

### 4.2. Входные данные

 - $a$, $b$ – границы изменения аргумента $x$;
 - $c$, $d$ – границы изменения аргумента $y$;
 - $\Delta x$ – шаг изменения аргумента $x$;
 - $\Delta y$ – шаг изменения аргумента $y$.

### 4.3. Выходные данные

 - таблица значений функции $Z$.

### 4.4. Математическая модель задачи

#### 4.4.1. Основные понятия

 - $x$, $y$ – аргументы функции;
 - $Z$ – значение функции.

#### 4.4.2. Формульная зависимость

$$Z = \frac { e^{ \sqrt{ 4 - x^2 - y^2 } } }{ x - y }$$

Область существования функции:

$$4 - x^2 - y^2 \ge 0, \quad x \ne y$$

### 4.5. Алгоритм решения задачи

1. Задать границы изменения аргументов и шаг табуляции.
2. Организовать вложенные циклы по $x$ и $y$.
3. Проверить область существования функции.
4. Если функция существует – вычислить значение $Z$, иначе вывести `(*)`.
5. Вывести результаты в виде таблицы.

      double z = exp(sqrt(4 - x * x - y * y)) / (x - y);
